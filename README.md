# HTML Resume Website
This is a single page resume website created using only HTML.

## Features
- Built using pure HTML (no CSS)
- Sections include About, Skills, Experience, Projects, Education, and Contact
- Clean and readable structure using semantic HTML tags

## Setup Steps
1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in a browser  
   OR  
   Open with Live Server in VS Code
